# Propost (TO-DO)

## Style

**Body**
```
- key:query:type
```

**Key**
```
- key:
```

**Query**
```
itemOne >> itemTwo
```

```
itemOne >> itemTwo @> {index: index, elem: elem}
```

```
itemOne >> itemTwo @> [index, elem]
```

```
itemOne >> itemTwo @> [elem]
```

**Type**
```
:type
```

- `string`
- `number`
- `array`
- `object`

## Operators
- **==**
- **~=**
- **&&**
- **||**
- **++**
- **\-\-**
- **%%**
- **XX**
- **@>**
- **>>**
- **!!**

## Methods
- **()**
- **[]**
- **{}**
