# Propost (TO-DO)

## Style

**Body**
```
- key:query:type
```

**Key**
```
- key:
```

**Query**
```
itemOne >> itemTwo
```

```
itemOne >> itemTwo @> {indexName, elemName, subItemOne >> subItemTwo}
```

```
itemOne >> itemTwo @> [subItemOne >> subItemTwo]
```

```
itemOne >> itemTwo @> subItemOne >> subItemTwo
```

```
itemOne == itemTwo
```

```
itemOne >> itemTwo == "Hello world"
```

**Type**
```
:type
```

- `string`
- `number`
- `array`
- `object`

## Operators / Methods
- **==**
- **~=**
- **||**
- **++**
- **\-\-**
- **%%**
- **XX**
- **@>**
- **>>**
- **!!**
- **[]**
- **{}**
- **""**
